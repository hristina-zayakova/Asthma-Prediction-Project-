# Asthma Diagnosis Prediction - Data Processing Pipeline

A professional-grade data processing pipeline for medical diagnosis prediction, specifically designed for asthma diagnosis using patient symptom and demographic data.

## Project Overview

This project provides a complete, production-ready data processing pipeline that transforms raw medical data into model-ready features for machine learning. The pipeline is specifically optimized for:

- **Medical diagnosis prediction tasks**
- **Imbalanced binary classification**
- **Mixed data types** (numeric, categorical, binary)
- **Reproducible and auditable processing**
